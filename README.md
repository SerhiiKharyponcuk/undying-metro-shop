# Undying Metro Shop

Адаптивна сторінка-посилання з анімованим фоном, мобільними ефектами та власною аватаркою магазину.

## Як відкрити

Відкрийте `index.html` у браузері або завантажте весь проєкт на будь-який статичний хостинг (GitHub Pages, Vercel, Netlify).

## Як додати посилання

Відкрийте `script.js` і вставте адреси в обʼєкт `LINKS`:

```js
const LINKS = {
  shop: "https://t.me/...",
  rent: "https://t.me/...",
  managers: "https://t.me/...",
  donate: "https://...",
  telegram: "https://t.me/...",
};
```

Якщо поле залишити порожнім, кнопка покаже акуратне повідомлення-заглушку.
